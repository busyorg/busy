## Quick Setup

```bash
git clone https://github.com/informationwar/newspeak.git
cd newspeak

yarn
yarn dev
```

## Production build

```bash
git clone https://github.com/informationwar/newspeak.git
cd newspeak

yarn
yarn build
yarn start
```
